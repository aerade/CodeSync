Задача: С нуля разработать полноценное, независимое нативное/десктопное приложение (не просто веб-обертку/WebView), которое будет дублировать, но качественно превосходить функционал моего текущего веб-проекта.

Контекст текущего проекта (CodeSync): > Мой текущий проект — это CodeSync, веб-IDE для совместной работы над кодом (интерфейс на русском языке). Текущий стек: фронтенд на React + Vite, сервер на Express + WebSocket, база PostgreSQL/Drizzle, а для real-time синхронизации используется Yjs.

Стек технологий для десктопа (предлагаемый):
Используй современный стек: Tauri (Rust) или Electron для бекенда/нативной оболочки, и React + Tailwind CSS + TypeScript для фронтенда. Для редактора сохраняем Monaco Editor, для синхронизации — Yjs. Если считаешь другой стек или архитектуру более подходящими под задачу нативного приложения — обоснуй.

Требования к функционалу («Тот же, но лучше»):

Базовые функции (перенос из веба): >    - Совместное редактирование файлов в реальном времени (через комнаты с приглашением по коду).

Встроенный Monaco-редактор и дерево файлов (создание/удаление файлов и папок).

AI-панель: чат, ревью кода, история изменений.

Терминал для запуска кода.

Восстановление версий файлов из истории и гостевой вход без аккаунта.

Прямой доступ к ФС: Работа с локальной файловой системой компьютера пользователя (а не только с виртуальным деревом на сервере).

Требования к UI/UX дизайну (Стиль: Cursor, Claude Code, Antigravity):
